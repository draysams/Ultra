Ultra is a visual, cross-platform SSH client that parses your terminal output to improve readability and show documentation on mouseover.

This brower-based demo (source code) only parses the bash command top (tested on Ubuntu 18.04.3 LTS), but it's forked from WebSSH2, so it can SSH into any server. I setup a demo server (67.205.180.23, username and password is demo) in case you're not comfortable SSH'ing into one of your own servers through my client. I'm not responsible for anything that people do to the demo server.

I'm working on a desktop version that parses more command outputs. Drop your email if you're interested, or (even better!) DM me on Twitter with any suggestions or questions.
